# ClipOval

This widget clip the child widget like ClipOval Widget in Flutter.

<b>The ClipOval</b> take clipper that is callback function and return <b>Rect</b> object.

This widget use <b>ClipPath</b> internally to clip the child widget

<br />

## Basic

<Source
  language='jsx'
  dark
  format={false}
  code={dedent`
  import { Container, ClipOval } from '@moonmoonbrothers/flutterjs';
  import { Rect } from '@moonmoonbrothers/flutterjs/types';

  ClipPath({
    clipper: (size) => 
      Rect.fromCenter({
        center: { x: size.width / 2, y: size.height / 2 },
        width: size.width / 2,
        height: size.height / 2
      }),
    child: Container({
      color: 'black'
    })
  })
`}
/>
